# Naming

Last updated: 2026-05-02

## Current candidate

Current project/product name candidate: `SteadyHearth`.

Why it fits:

- calm and emotionally warm
- not clinical or diagnosis-oriented
- suggests steadiness without promising medical outcomes
- broad enough to support CBT, journaling, check-ins, and later self-help tools

Risks and checks:

- `hearth` may be misread as home/fireplace rather than mental wellness
- the spelling must be kept consistent: `SteadyHearth`, not `SteadyHeart`, unless the project intentionally changes direction
- quick web search on 2026-05-02 showed some unrelated uses of `Steady Hearth`, including music and a firewood rack product, but no obvious mental-health app result in the quick search
- before buying a domain, changing package IDs, or submitting to app stores, run proper trademark, domain, GitHub, npm, app-store, and social-handle checks

## Temporary domain

Use `https://yoshevbot.uk` for the temporary web test. A final domain can be chosen after the name is approved.

Possible future domain patterns:

- `steadyhearth.org` if available and nonprofit positioning is central
- `steadyhearth.app` if the app identity is primary
- `steadyhearth.com` if available and not too expensive

Do not change the Android package name until the final app name decision is made.
